Arrays and Strings

Arrays

An array is a linear data structure that collects elements of the same data type and stores them in contiguous and adjacent memory locations.

Array Operations

# Creating an array
numbers = [1, 2, 3, 4, 5]

# Accessing elements
first_element = numbers[0]  # 1
last_element = numbers[-1]  # 5

# Modifying elements
numbers[0] = 10  # [10, 2, 3, 4, 5]

# Adding elements
numbers.append(6)  # [10, 2, 3, 4, 5, 6]
numbers.insert(0, 0)  # [0, 10, 2, 3, 4, 5, 6]

# Removing elements
numbers.pop()  # [0, 10, 2, 3, 4, 5]
numbers.remove(10)  # [0, 2, 3, 4, 5]

2D Arrays (Matrices)

# Creating a 2D array
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

# Accessing elements
element = matrix[1][2]  # 6

# Iterating through a 2D array
for row in matrix:
    for element in row:
        print(element)

Strings

A string is a sequence of characters. In Python, strings are immutable, meaning they cannot be changed after creation.

String Operations

# Creating strings
text = "Hello, World!"
name = 'Python'

# String concatenation
greeting = "Hello, " + name  # "Hello, Python"
greeting = f"Hello, {name}"  # Using f-strings

# String methods
text.upper()  # "HELLO, WORLD!"
text.lower()  # "hello, world!"
text.split()  # ["Hello,", "World!"]
text.replace("World", "Python")  # "Hello, Python!"

String Manipulation

# Reversing a string
def reverse_string(s):
    return s[::-1]  # Using slicing

# Checking palindrome
def is_palindrome(s):
    return s == s[::-1]

# Counting characters
def count_chars(s):
    char_count = {}
    for char in s:
        char_count[char] = char_count.get(char, 0) + 1
    return char_count
Common String Problems:
  • String reversal
  • Palindrome checking
  • Anagram detection
  • Longest substring without repeating characters
  • String compression

Time Complexity Analysis

  • Array access: O(1)
  • Array insertion/deletion: O(n)
  • String concatenation: O(n)
  • String slicing: O(k) where k is the size of the slice
